# Tableau-Projects
## 🔹 Project Overview  
- This **HR Analytics** project explores employee data to analyze key **HR metrics**, including attrition rates, hiring trends, and performance evaluation.  

## 🔹 Tools Used  
- **Tableau** (for data visualization)  
- **Excel / CSV** (For data)
- **SQL** (optional, for preprocessing)  

## 🔹 Insights & Key Findings  
- ✔ **Attrition Analysis** – Identifies reasons why employees leave.  
- ✔ **Employee Performance** – Analyzes trends in performance ratings.  
- ✔ **Hiring Trends** – Tracks recruitment patterns over time.  
- ✔ **Department-wise Analysis** – Compares HR metrics across different teams.
